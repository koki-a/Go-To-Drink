# Go To Drink

URL:http://gotodrink.work <br>

## アプリ概要
札幌すすきのに特化した飲食店の共有サイトです。<br>
コロナウィルスの影響で飲みに行きづらいのがきっかけと、
元々飲みに行くが好きで、他人にお勧めのお店を紹介することも多かったので、自身のデータベースのような意味も含め作成しました。

# 使用技術
- PHP 7.3
- Laravel 6.20
- Javascript
- Vue.js 2.6.11
- Sass
- MySQL
- Apache
- PHPUnit
- Debugbar
- Google Maps API
- AWS
  - VPC
  - EC2
  - RDS
  - Route53

# 機能一覧
- ログイン・ログアウト
- CRUD機能（ユーザー・店舗登録、編集、削除）
- マイページ
- いいね処理（非同期通信処理)
- 複数条件検索機能
