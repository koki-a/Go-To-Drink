# Go To Drink
http://gotodrink.work <br>
飲食店の共有サイトです。<br>
コロナウィルスの影響で飲みに行きづらいのがきっかけと、
元々飲みに行くが好きで、他人にお勧めのお店を紹介することも多かったので、自身のデータベースのような意味も含め作成しました。
